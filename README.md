# Playlist Manager
A playlist manager using Spotify Web API.

## Features
* Create, edit, and unfollow a playlist
* Add and remove tracks to playlist
* Search tracks

## Features to add
* Play a playlist or a track
* Queue a track
* Custom playlist cover image

## Technology used
* Next.JS
* Tailwind
* Framer Motion
* React Icons
* Axios
* Spotify Web API
